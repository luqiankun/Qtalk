# 客户端接收到的报文格式
```json
{
    "header":0,
    "msg":"string",
    "return":1,
    "headImg":"图像的base64编码",
    "from":"33239898"
}
```
#### 释意 ：
- “header”:
  - 1: 登录结果
  - 2：注册结果
  - 3：发送消息给他人是否成功
  - 4：连接服务器是否成功
  - 5：接收到最新的在线用户列表
  - 6：收到来自他人的消息
  - 7: 头像上传成功
- “msg”:
  - 当 $header$等于5时，其内容为所有用户的名称和头像，为json数组格式 
      > "msg":[["name1","headImg1"],[...,...]]
  - 当$header$等于6时，其内容为接收到的消息具体内容
  - 当$header$等于7时，其内容为用户头像base64编码

# 服务器接收到的报文格式
```json
    "header":1,
    "id":"userName",
    "pwd":"pwd",
    "msg":"string",
    "sendto":"33239899"
```
- 其内容具体含义和服务器类似
- 特殊：当$header=5$时为用户上传头像，此时msg内容为头像的base64编码
